# Takasimba

A hypertrophy-focused lifting app to track your workouts, volume, and progress.

## Features

- **Workout logging** – Log sets, reps, and weight for each exercise
- **Exercise library** – 30+ common hypertrophy exercises across all muscle groups
- **Volume tracking** – Automatic calculation of total volume (sets × reps × weight)
- **Workout history** – View and manage past workouts
- **Local storage** – Data persists in your browser

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- React 18 + TypeScript (TSX)
- date-fns

## Muscle Groups

Chest, Back, Shoulders, Biceps, Triceps, Quads, Hamstrings, Glutes, Calves, Core
